# Places API

Return a number places nearby by providing latitute, longitude and a customer name

### Before anyting add API keys provided to the configuration file (conf.js)

```
module.exports = {
  customerInfo:[
    {
      name :'Sunrise Bank',
      api_key: process.env.NODE_ENV ==='test_nokey' ? 'test' : '', // ADD KEY HERE
      language :'en',
      type: 'atm',
      response :'xml',
      request_no: 200
    },
    {
      name :'Happy Credit Union',
      api_key: process.env.NODE_ENV ==='test_nokey' ? 'test' : '', // ADD KEY HERE
      language :'es',
      type: 'bank',
      response :'json',
      request_no: 20
    },
    {
      name :'Paris FCU',
      api_key: process.env.NODE_ENV ==='test_nokey' ? 'test' : '',// ADD KEY HERE
      language :'fr',
      type: 'all',
      response :'json',
      request_no: 5
    },
  ],
  placeAPI:'https://maps.googleapis.com/maps/api/place/nearbysearch/',
  defaultReq:20,
  maxRadiusTry: 5,
  minRadius:  1500,
  maxRadius:  50000
};

```

### Install

Get dependencies

```
npm install
```

### Run server

```
> node server.js

Starting in port 3000
```
